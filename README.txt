Baby Skin – Landing Page (com Widget de Vendas)
================================================

Como usar
---------
1) Envie **todos** os arquivos deste pacote para o seu repositório (mantenha a pasta `assets/`).
2) Publique na Vercel (Framework: Other).

Arquivos
--------
- index.html          → página completa com widget e paleta integrada
- logo-sdb.png        → logo com fundo transparente e brilho apenas nela
- assets/baby-skin-hero.webp  → imagem hero otimizada (fallback .jpg)
- assets/baby-skin-hero.jpg   → fallback jpg

Widget
------
- Ajustes rápidos no final do `index.html` no objeto `CONFIG`.
- API de testes no console: `BabySkinSales.pause()`, `resume()`, `setStock(200)`, `buy(2)`.
